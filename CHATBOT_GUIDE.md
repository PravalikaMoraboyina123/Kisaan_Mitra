# 🤖 Kisaan Mitra Chatbot Guide

## Overview
The Kisaan Mitra Chatbot is a multilingual AI assistant that provides personalized farming advice, answers agricultural questions, and helps farmers make better decisions.

## Features

### 🌍 Multilingual Support
- **English**: Ask questions in English for responses in English
- **हिंदी (Hindi)**: Ask in Hindi for Hindi responses
- **తెలుగు (Telugu)**: Ask in Telugu for Telugu responses

### 💬 Topics Covered
The chatbot can help with:
- **Crop Management**: Selection, sowing, fertilizers, and harvest timing
- **Soil & Water**: Soil health, irrigation techniques, water management
- **Pest & Disease Control**: Identification and treatment strategies
- **Weather**: Understanding weather patterns and their impact on crops
- **Market Prices**: Mandi prices and selling strategies
- **Government Schemes**: Information about farmer subsidies and support
- **Sustainable Farming**: Eco-friendly practices and techniques

## How to Use

### Starting a Chat
1. Click on the **💬 Chat Assistant** button in the navigation menu
2. The chat interface will open with your message input box at the bottom

### Asking Questions
1. Type your question in the message input field
2. You can ask in **English**, **हिंदी (Hindi)**, or **తెలుగు (Telugu)**
3. Click the **Send** button or press Enter to submit
4. The Kisaan Mitra assistant will respond in the same language you used

### Chat History
- All your messages and responses are saved in the current session
- View statistics in the sidebar:
  - **Total Messages**: Total conversation count
  - **Your Messages**: Number of questions you asked
  - **My Responses**: Number of replies from Kisaan Mitra

### Managing Chat
- **🗑️ Clear Chat**: Delete all messages and start fresh
- **💾 Export Chat**: Download your entire conversation as a text file
- **📥 Download Button**: Save the chat history for future reference

## AI Provider Settings

The chatbot works with both:
- **Local (Ollama)**: For offline, free AI responses
- **Cloud (BYOK)**: For advanced models like GPT-4, Claude, Gemini

Configure your preferred provider in the sidebar before chatting.

## Tips for Better Responses

1. **Be Specific**: Include your location, crop type, and problem details
2. **Ask Follow-ups**: Ask clarifying questions to get more detailed advice
3. **Mix Languages**: Feel free to switch languages in the same conversation
4. **Export & Share**: Save useful responses to share with other farmers
5. **Cross-Reference**: Compare chatbot advice with other features like Crop Advisor

## Troubleshooting

### No Response
- Check your AI provider settings in the sidebar
- Ensure Ollama is running (if using local) or API key is valid (if using cloud)
- Try a simpler question first

### Language Issues
- Ensure you're typing in the correct language keyboard
- The chatbot will respond in the language you use for each message
- You can mix languages in the same conversation

### Chat History Lost
- Session chat history is cleared when you refresh the page
- Use the **Export Chat** feature to save important conversations
